📋 Setup Guide — Connect to Google Sheets
Step 1 — Create a Google Sheet
Go to sheets.google.com → New Spreadsheet. Name it anything.
Step 2 — Add the Apps Script
In your sheet: Extensions → Apps Script → delete all existing code → paste the full contents of Code.gs → Save (Ctrl+S).
Step 3 — Deploy as Web App
Click Deploy → New Deployment → Select type: Web App
Execute as: Me | Who has access: Anyone
Click Deploy → click through the Google warning (click Advanced → Go to app) → Allow
Copy the Web App URL (looks like https://script.google.com/macros/s/ABC…/exec)
Step 4 — Initialize Sheets
Paste your URL in the Config section below and click Connect. It will auto-create all tabs (Users, Items, Quotations, Folders, Config).
Step 5 — Set your password
Open your Google Sheet → Users tab → edit the password column. Changes take effect immediately. Default: admin / admin123
Step 6 — Gemini API Key (for PDF import)
Go to aistudio.google.com → Get API Key → Create API Key. Paste it in Config below or in the Sheet's Config tab.
Step 7 — Host on GitHub Pages (free)
github.com → New public repo → upload this file as index.html
Settings → Pages → Source: main branch → Save.
Your live URL: https://yourusername.github.io/repo-name
